Usage Guidelines

Clear Space

Maintain clear space around all logos equal to the height of the "H" in HQ. No other elements should intrude on this space.

Minimum Size

Full wordmark: minimum 120px wide. Short mark (FOHQ): minimum 60px wide. Below these sizes, legibility suffers.

Color Usage

Use the gold accent on "HQ" when possible for brand recognition. Fall back to monochrome versions when color printing isn't available or on complex backgrounds.

Background Rules

Use navy/dark logo on white or light gray backgrounds. Use white logo on navy or dark backgrounds. Avoid placing logos on busy images or patterns.

Don't

Don't stretch, rotate, add effects, change colors, or rearrange logo elements. Don't place on low-contrast backgrounds.

File Formats

Use SVG for web and digital. Use PNG with transparent background for documents and presentations. Request vector files (AI/EPS) for print production.
