# How to Export Podcast Cover Art to PNG

The cover art is provided as an SVG file (`cover-3000x3000.svg`) for maximum quality and editability. Here's how to export it to PNG for podcast platforms.

## Option 1: Using Figma (Easiest)

1. Open Figma (figma.com)
2. Create a new design file
3. File → Import → Select `cover-3000x3000.svg`
4. Select the imported frame
5. In the right panel, under "Export", click **+**
6. Set format to **PNG** and scale to **1x**
7. Click **Export**

## Option 2: Using Browser + Screenshot

1. Open `cover-3000x3000.svg` in Chrome/Safari
2. Zoom to 100%
3. Take a high-quality screenshot
4. Crop to exact dimensions

## Option 3: Using Inkscape (Free)

1. Download Inkscape (inkscape.org)
2. Open `cover-3000x3000.svg`
3. File → Export PNG Image
4. Set dimensions to 3000x3000
5. Click Export

## Option 4: Online Converter

1. Go to cloudconvert.com or svgtopng.com
2. Upload `cover-3000x3000.svg`
3. Set output size to 3000x3000
4. Download PNG

## Platform Requirements

| Platform | Size | Format |
|----------|------|--------|
| Apple Podcasts | 3000x3000 (min 1400x1400) | JPEG or PNG |
| Spotify | 3000x3000 (min 640x640) | JPEG or PNG |
| Google Podcasts | 3000x3000 | PNG |
| YouTube | 3000x3000 | PNG recommended |

## Notes

- Always export at full 3000x3000 resolution
- Use PNG format to preserve quality (no JPEG compression artifacts)
- The SVG source file can be edited if you need to make changes
